回到开头的问题 sys.exit(0) 与 sys.exit(1) 的区别，都会raise SystemExit异常，但是区别在于程序的返回码。

sys.exit(0) 为正常退出，echo $？，结果为0

sys.exit(1) 为非正常退出，echo $， 结果为1
